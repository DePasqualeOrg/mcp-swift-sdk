# MCP Swift SDK

Swift implementation of the Model Context Protocol.

## Related Repositories

The parent directory contains related MCP repositories:

- `modelcontextprotocol`
  - Schema: `schema`
  - Specification: `docs/specification`

- `mcp-typescript-sdk` - TypeScript SDK (most up to date, use as reference)
- `mcp-python-sdk` - Python SDK (most up to date, use as reference)

## Documentation

- `planning-docs/architecture-analysis.md` - Detailed architectural analysis comparing Swift with TypeScript/Python SDKs; implementation plan for HTTP server support via request capture pattern
- `planning-docs/stallent-implementation.md` - Analysis of Stallent's alternative HTTP server implementation approach
- `planning-docs/swift-ts-python-sdk-comparison.md` - Comparison of Swift, Python, and TypeScript SDKs; identifies gaps and strengths
- `planning-docs/swift-sdk-protocol-gaps.md` - Detailed analysis of missing protocol features and implementation plan to reach version 2025-11-25
